# HDB Airconds Company Profile Website

Website company profile untuk bisnis **HDB Airconds** yang menyediakan layanan servis dan instalasi AC.

🌐 Live Website  
https://hdbairconds.id

---

## About The Project

Project ini merupakan website company profile berbasis **static website** yang dibuat untuk membantu perusahaan dalam menampilkan informasi layanan dan meningkatkan kehadiran bisnis secara online.

Website ini menampilkan informasi tentang layanan servis AC, profil perusahaan, serta kontak yang dapat dihubungi oleh pelanggan.

---

## Technologies Used

- HTML5
- CSS3
- JavaScript

---

## Features

- Home Page
- Company Profile
- Service Information
- Contact Information
- Responsive Layout
